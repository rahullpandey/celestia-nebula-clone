# Celestia - Nebula Clone Challenge

Mobile-first astrology app inspired by Nebula, Starcrossed, and Star Walk 2.

## Features

- Daily, weekly, and monthly horoscope feed with love, energy, and luck scores
- Birth chart generator with animated zodiac wheel, planet glyphs, and house breakdown
- Compatibility report with dynamic score and synastry-style relationship guidance
- AI astrologer chat simulation with chart-aware responses for love, career, and timing
- Personalized sun, moon, and rising profile updates from birth date, time, and place

## Run Locally

```bash
python3 -m http.server 8088
```

Open `http://127.0.0.1:8088`.

## Demo Walkthrough

1. Start on the horoscope feed and switch between Daily, Weekly, and Monthly.
2. Tap Chart, update birth date/time/place, and generate the natal chart.
3. Tap Match, choose a partner sign, and recalculate compatibility.
4. Tap Chat and ask a relationship, career, or daily-energy question.
